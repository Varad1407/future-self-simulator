## 🚀 Features
- ML-based prediction (Random Forest)
- Current vs Ideal comparison
- Future simulation (30 / 90 / 180 days)
- Risk analysis & insights
- User history tracking
- Interactive dashboard (Streamlit)

---

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

---

## ⚙️ How It Works
1. User inputs daily habits
2. ML model predicts discipline score
3. App simulates future outcomes
4. Displays insights and growth projections
